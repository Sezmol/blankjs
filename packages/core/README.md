# @blankjs/core

Headless form primitives for React. Zero dependencies.

The hooks behind [@blankjs/react](https://www.npmjs.com/package/@blankjs/react): controllable state, collections, and field validation built on the Constraint Validation API.

```bash
npm install @blankjs/core
```

```tsx
import { useControllableState, useFieldRoot, useCollection } from "@blankjs/core";
```

Use it to build your own components on the same validation and state machinery, or install `@blankjs/react` for the styled components.
